Welcome to Cohort Analysis Dashboard, or CAD for short!

CAD extends on the UI Framework for PM4PY to allow the applications of methods of Cohort Analysis to Process Data with a focus on data derived from the Medical Domain.

CAD's goal is to create a Dashboard for users working with process data, providing them with informative views on the process and creating new insights for the process owner.

CADs aim is to allow users to analyse processes by creating event groups and view their relation to the process by means of learning about the number of concurrently running events and the time intervals each group of events is happening in.
