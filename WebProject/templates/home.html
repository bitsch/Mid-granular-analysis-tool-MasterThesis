Welcome to Mid granular analysis tool Dashboard!

This dashboard extends on the UI Framework for PM4PY to allow the application of mid granular analysis on Data for the purpose of thesis.

The goal is to create a Dashboard for users working with process data, providing them with means to do the mid granular level analysis.

The dashboard contains simplified version of the analysis to show the working of the project, more detailed experiment is performed in jupyter notebook available on Github .
